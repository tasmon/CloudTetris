# CloudTetris

A Tetris clone built for Cloud Phone — designed for tiny 240x320 feature phone screens with full keyboard/T9 navigation (no touch required). Three game modes, four retro-inspired themes, and softkey-driven menus.

Controls: 4/6 move · 2/5 rotate · 8 hard drop · 0 pause

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/CloudTetris`  
4. Upload icons if required.  
5. Save and refresh — **Cloud Tetris** will appear on your CloudPhone.
